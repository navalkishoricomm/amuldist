## Notes

- Admin can create distributors/retailers, activate/deactivate, edit profiles, delete users, and view stats.
- Distributors/Retailers can edit their profile once; further edits require Admin.
- Web app runs in `AmulDistAppClient` with `npm run web`, API runs in `server` with `npm run start`.
